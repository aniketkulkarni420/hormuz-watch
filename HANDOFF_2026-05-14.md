## 3. Architecture

```
Browser (vanilla JS + Leaflet — MAP CURRENTLY BROKEN, see §11)
  ↓
Cloudflare Pages Functions (~20 endpoints at functions/api/*.js)
  ↓
KV (15+ keys) + D1 (snapshots, commentary, feature_events, subscribers)

```

Single HTML file (~3200 lines). No build chain. Vanilla JS + Leaflet 1.9.4 + Manrope + JetBrains Mono.

### Multi-source resilience pattern (key learning)
For unreliable free sources, use:
1. 2-3 independent sources with different infra (HTTP API + Playwright + RSS)
2. Sanity bounds per signal
3. Median + spread → confidence rating (≤5% spread = high)
4. Write to KV with breakdown
5. Frontend shows median with confidence indicator
6. Fallback chain in API: Tier 0 (cross-verified) → Tier 1 (single source) → Tier 3 (ETF proxy)

seismic 0.03 · weather 0.03 · ofac 0.10 · currency 0.06 · news 0.05 ·

Score → verdict: <1.0 NORMAL · 1.0-2.0 ELEVATED · 2.0-3.0 HIGH · ≥3.0 CRITICAL

- OFAC: new Iran designation in last 48h
- Currency: IRR spread >150%
- News: 40+ headlines in 24h
- Aircraft: military count >5
