The control room redesign — three versions

Built against the "digital control room" framing from your shared document. Sidebar nav (the spine), no scroll on overview, 3-7 tiles max, 24px typography ceiling, strict 4/8pt grid, stacked lists in dense areas, dark-mode elevation via lighter backgrounds (no shadows), semantic colour, and explicit personalisation for the Indian-investor persona we serve. Same single dataset rendered three ways so you can pick the density and persona-bias that fits.

Version 1
Strict Control Room
Maximum doc adherence. Sidebar spine. Six equal tiles in a 3×2 grid. No decoration, no charts beyond a 3-px progress strip or 6-px zone bar. Every tile fits on one screen. Stacked lists inside tiles instead of nested cards.
Applies: single-screen no-scroll · 6 metric tiles · ≤22px type · 24px gutters · stacked lists · no gauges · no shadows · top-left = answer (Verdict)
